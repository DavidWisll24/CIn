# We is Us

O contexto do código é averiguar se no aglomerado de caminhões dispóniveis foi possível acomodar os itens pedidos por Carol

Primeiro, seram recebidos os pedidos de itens de Carol, separados por vírgula e espaços.
Depois, seram pedidios itens que estão nos caminhões, tendo um número indefinido de itens e de caminhões, e só para de receber ao receber "--"

Após receber as informações, é verificado de 1 emm 1 caminhão se algum item foi recebido, qual item e qual falta.
Se no fim receber todos, print a mensagem de sucesso, senão, printe a mensagem de erro

OBS: se ela pedir arroz 3 vezes, tem que ter arroz 3 vezes nos caminhões para sair da lista e tem que printar 3 arroz
